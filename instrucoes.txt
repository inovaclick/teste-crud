##instruções para criação do banco de dados.

Criar o banco dados como user root em localhost
Alterar a senha do user root para "umasenhaaleatoria"
"$ sudo mysql -uroot -p"
"$ ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY 'umasenhaaleatoria';"

Caso não queira realizar as configurações acima basta mudar no arquivo db.config.js para a configuração que desejar, respeitando
a tabela que será descrita abaixo.

create database `inovaclick_crud`;

use inovaclick_crud;

CREATE TABLE IF NOT EXISTS `users` (
	`id` BIGINT UNSIGNED AUTO_INCREMENT,
    `name` VARCHAR(255) NOT NULL,
    `address` VARCHAR(255) NOT NULL,
    `cpf` VARCHAR(255) NOT NULL,
    `email` VARCHAR(255) NOT NULL,
    primary key(`id`)
);

Após a criação é possível testar as apis tanto na spa quanto no postman.

##instruções para execução dos programas.

Irá ocupar duas abas de terminais, na primeira entre na pasta back-end e instale as dependências com npm install, após as
dependências estarem rode o comando npm start, o servidor estará aberto na porta 5000.

No segundo terminal entre na pasta front-end instale as dependências com npm install, e rode o comando npm start, processo 
estará rodando na porta 8080.
